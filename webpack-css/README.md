# webpack处理css文件  --> style标签引入css文件
1、安装webpack
`npm install webpack webpack-cli --save-dev`
2、安装css-loader和style-loader
`npm install --save-dev css-loader style-loader`
3、安装plugin
`npm install --save-dev html-webpack-plugin`
